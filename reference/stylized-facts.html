Returns & Stylized Facts

Reference · print-friendly cheat sheet · Unit 004

Returns: definitions

simple: R_t = (P_t − P_{t−1}) / P_{t−1}   ·   log: r_t = ln(P_t / P_{t−1})

Log returns are additive across time (r over k periods = sum of r's) and symmetric around 0; simple returns are additive across assets (portfolio return = weighted sum). Quants default to log returns for time-series modeling. For small moves r_t ≈ R_t.

The stylized facts (Cont 2001)

Empirical regularities that hold across markets, assets, and eras. Any model of returns must respect them.

FactWhat you observeKills the assumption that…
Fat tailsKurtosis ≫ 3; extreme moves far more common than a Gaussian predictsreturns are normal
No linear autocorrelationcorr(r_t, r_{t+k}) ≈ 0 for k ≥ 1 (beyond microstructure lags)you can predict direction from past returns alone
Volatility clusteringcorr(\|r_t\|, \|r_{t+k}\|) > 0 and decays slowly — big moves follow big movesvolatility is constant (homoskedastic)
Leverage effectnegative returns raise future volatility more than positive onesvol responds symmetrically to up/down moves
Aggregational Gaussianityas the horizon grows (daily→monthly), the distribution looks more normalthe shape is scale-invariant
Gain/loss asymmetrylarge drawdowns sharper than equally large run-ups (equities)the series is time-reversible

Measuring them (Python)

import numpy as np, pandas as pd
r = np.log(px).diff().dropna()          # log returns

r.kurtosis()                            # fat tails: ≫ 0 (excess kurtosis)
r.autocorr(lag=1)                       # ~0 for raw returns
r.abs().autocorr(lag=1)                 # > 0: volatility clustering
(r.rolling(21).std()).plot()            # clustering, visually
Why it matters. Fat tails mean risk models built on the normal distribution systematically underestimate tail losses. Zero linear autocorrelation is why naïve "yesterday went up so buy" strategies fail — yet nonlinear and cross-sectional structure is where alpha hides. Volatility clustering is the entire reason GARCH and realized-volatility models (Units 023–025) exist.
Trap. Reporting a Sharpe ratio or a t-stat as if returns were IID normal. Fat tails + volatility clustering inflate apparent significance. This is the first crack through which overfit "edges" leak in — the whole of Year 2 Q2 exists to close it.
